# Project 2 - Exploratory Data Analysis (EDA)

## Overview

This project was completed as part of the DecodeLabs Data Analytics Internship Program. The objective of this project was to perform Exploratory Data Analysis (EDA) on an e-commerce dataset to identify patterns, trends, distributions, and meaningful business insights using Microsoft Excel and Pivot Tables.

---

## Objectives

* Perform descriptive statistical analysis
* Identify trends and patterns in the dataset
* Analyze product performance and revenue
* Evaluate payment methods and order statuses
* Analyze customer referral sources
* Generate meaningful business insights

---

## Dataset Information

The dataset contains approximately 1,200 e-commerce order records with the following attributes:

* OrderID
* Date
* CustomerID
* Product
* Quantity
* UnitPrice
* PaymentMethod
* OrderStatus
* ReferralSource
* TotalPrice

---

## Tasks Performed

### 1. Descriptive Statistics

Calculated:

* Count
* Mean
* Median
* Minimum
* Maximum

for the following columns:

* Quantity
* UnitPrice
* TotalPrice

---

### 2. Product Analysis

Created Pivot Tables to identify the most frequently ordered products.

**Key Finding:** Printer was the most frequently ordered product with 181 orders.

---

### 3. Revenue Analysis

Analyzed total revenue generated by each product category.

**Key Finding:** Chair generated the highest revenue, followed closely by Printer and Laptop.

---

### 4. Payment Method Analysis

Analyzed customer payment preferences using Pivot Tables.

**Key Finding:** Online payment was the most preferred payment method.

---

### 5. Order Status Analysis

Evaluated order processing outcomes.

**Key Finding:** Cancelled orders represented the largest order status category.

---

### 6. Referral Source Analysis

Analyzed customer acquisition channels.

**Key Finding:** Instagram generated the highest number of customer referrals.

## Key Insights

* Printer was the most frequently ordered product.
* Chair generated the highest overall revenue.
* Online payment was the most preferred payment method.
* Instagram was the leading referral source.
* Customer demand was relatively balanced across products and services.
